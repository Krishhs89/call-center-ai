  • Fallback to raw text if Whisper unavailable

  • Uses configurable LLM (Claude/GPT-4/Gemini)
  • Extracts: summary, key_points, action_items, customer_issue, resolution_status
  • Uses structured output with function calling
  • Default: Claude 3.5 Sonnet
  • Methods: process(), _build_prompt()

quality_score_agent.py
  • Multi-dimensional QA scoring (0-100 total)
  • Scores: empathy(0-25), professionalism(0-25), resolution(0-25), compliance(0-25)
  • Provides strengths and improvement suggestions
  • Uses structured output for reliable scoring
  • Methods: process(), _build_prompt(), validate_scores()

routing_agent.py
  • Manages state transitions and routing logic
  • Handles error recovery and fallbacks
  • Logs workflow progress (LangSmith-compatible)
  • Methods: route_intake(), route_transcription(), route_summarization(), etc.

API USAGE EXAMPLE
================================================================================

from agents.intake_agent import IntakeAgent
from workflow.langgraph_flow import create_workflow, run_workflow
from evaluation.benchmark import BenchmarkRunner

# Process input
intake = IntakeAgent()
call_input = intake.process(transcript_text="Agent: Hello. Customer: Hi.")

# Run workflow
workflow = create_workflow(llm_name="claude")
result = run_workflow(workflow, call_input)

# Access results
print(result.summary.summary)
print(result.qa_score.overall_score)

# Run benchmark
benchmark = BenchmarkRunner()
bench_result = benchmark.run_full_benchmark(
    call_id=result.call_id,
    transcript=result.transcript.transcript
)
comparison = benchmark.compare_results(bench_result)
print(comparison['fastest_model'])
